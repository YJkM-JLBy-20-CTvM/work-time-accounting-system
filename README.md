# Система учёта рабочего времени сотрудников

## Описание проекта

Данная информационная система предназначена для автоматизации учёта рабочего времени сотрудников организации.  
Программа позволяет фиксировать время прихода и ухода сотрудников, управлять данными сотрудников, а также формировать отчёты за выбранный период.

Система реализована с использованием:
- Python
- PyQt5 (графический интерфейс)
- PostgreSQL (база данных)
- psycopg2 (подключение к БД)

---

## Основные возможности

### Авторизация
- Вход по логину и паролю
- Разграничение ролей:
  - Администратор
  - Сотрудник

---

### Функции сотрудника
- Отметка времени прихода
- Отметка времени ухода
- Просмотр своего рабочего времени
- Смена пароля

---

### Функции администратора
- Просмотр рабочего времени сотрудников
- CRUD-операции:
  - Сотрудники
  - Отделы
  - Должности
- Формирование отчётов:
  - по сотруднику
  - по отделу
  - за период времени

---

### Отчёты
- Генерация отчётов в формате `.docx`
- Табличное представление данных
- Расчёт отработанных часов

---

## Структура базы данных

### Таблицы:
- `departments` — отделы
- `positions` — должности
- `employees` — сотрудники

## Установка и запуск

### 1. Установить зависимости

```bash
pip install PyQt5 psycopg2 python-docx
```

### 2. Создать базу данных

В PostgreSQL (через pgAdmin или psql) создать БД:

```bash
employee_time_tracking
```

### 3. Настроить подключение

Файл db_connection.py:

```bash
conn = psycopg2.connect(
    host='localhost',
    dbname='employee_time_traking',
    user='postgres',
    password='12345'
)
```

### 4. Инициализация БД

```bash
python db_init.py
```

### 5. Заполнение текстовыми данными

```bash
python db_insert.py
```

### 6. Запуск программы

```bash
python main.py
```
