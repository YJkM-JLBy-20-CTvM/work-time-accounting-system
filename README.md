# Система учёта рабочего времени сотрудников

## Описание проекта
Данная информационная система предназначена для автоматизации учёта и контроля рабочего времени сотрудников компании.

## Функционал
- Авторизация пользователей
- Фиксация времени прихода и ухода сотрудников
- Просмотр рабочего времени
- Управление сотрудниками
- Формирование отчётов

## Технологии
- Python
- PyQt5
- PostgreSQL
- psycopg2

## Установка и запуск

1. Клонировать репозиторий:

git clone https://github.com/YJkM-JLBy-20-CTvM/work-time-accounting-system

2. Установить зависимость:

pip install -r requirements.txt

3. Запустить программу:

python main.py

## Структура проекта
- database — работа с БД
- models — модели данных
- ui — пользовательский интерфейс
